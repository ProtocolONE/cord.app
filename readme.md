### Build Tools ###
 * MSVC 2013-2017 runtime 120_xp
 * conan.io (>= 1.7.2)
 * .NET 3.5

### Build ###
Clone application

    git clone ssh://git@git.syncopate.local:7999/pon/launcher.git

Init submodules

    submodule.init.cmd
    
Install dependecies

    install.deps.cmd
    
Build by MSVC Launcher.sln
